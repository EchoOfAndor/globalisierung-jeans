Projekt: Die Reise einer Jeans – Globalisierung in der Textilindustrie

Diese Website ist ein rein nicht-kommerzielles Schulprojekt für das Fach Geographie an der Freien Montessorischule Huckepack.
Um die komplexen Inhalte der globalen Lieferkette anschaulich aufzubereiten, wurden stilistische Elemente aus dem Videospiel Helldivers 2 als humorvolles Design-Thema ("Gamification") integriert. Alle Rechte an diesen fiktiven Elementen liegen bei den jeweiligen Urhebern; es besteht keinerlei Absicht zur kommerziellen Nutzung oder Markenrechtsverletzung.
